**Step A — Add 4 new queries to the `Promise.all` in `loadData()`:**

Import additions at top of file: add `gte, and` to the drizzle-orm import. Add `import { KEYWORDS, extractKeywordCounts, profileKeywordSet } from "@/lib/analytics/keywords";`.

1. **matchQualityTrend** — daily avg fitScore, tier 1-3, last 30 days:
   ```ts
   db.select({
     date: sql<string>`to_char(date_trunc('day', ${schema.jobs.discoveredAt}), 'YYYY-MM-DD')`.as("date"),
     avgScore: sql<number>`round(avg(${schema.jobs.fitScore}::numeric), 1)::float`,
   })
     .from(schema.jobs)
     .where(and(
       inArray(schema.jobs.tier, [1, 2, 3]),
       gte(schema.jobs.discoveredAt, sql`now() - interval '30 days'`),
     ))
     .groupBy(sql`date_trunc('day', ${schema.jobs.discoveredAt})`)
     .orderBy(sql`date_trunc('day', ${schema.jobs.discoveredAt})`)
   ```

2. **applicationsPerDay** — applications sent per day, last 90 days:
   ```ts
   db.select({
     date: sql<string>`to_char(date_trunc('day', ${schema.applications.appliedAt}), 'YYYY-MM-DD')`.as("date"),
     count: sql<number>`count(*)::int`,
   })
     .from(schema.applications)
     .where(and(
       isNotNull(schema.applications.appliedAt),
       gte(schema.applications.appliedAt, sql`now() - interval '90 days'`),
     ))
     .groupBy(sql`date_trunc('day', ${schema.applications.appliedAt})`)
   ```
   (No orderBy needed — we map into a fixed 90-day grid.)

3. **jdTextsForKeywords** — tier 1-3 jdText for keyword extraction:
   ```ts
   db.select({ jdText: schema.jobs.jdText })
     .from(schema.jobs)
     .where(and(
       inArray(schema.jobs.tier, [1, 2, 3]),
       isNotNull(schema.jobs.jdText),
     ))
   ```
   (Entire tier-1-3 corpus. If the corpus grows beyond ~2k rows later, cap via `.limit(2000)` — for now leave uncapped; current DB has ~508 total jobs.)

4. **profileRow** — single profile row for strengths matching:
   ```ts
   db.select({
     headline: schema.profile.headline,
     roles: schema.profile.roles,
     toolStack: schema.profile.toolStack,
     achievements: schema.profile.achievements,
     industries: schema.profile.industries,
   })
     .from(schema.profile)
     .limit(1)
   ```

Destructure all 4 new rows from the `Promise.all` alongside the existing 9.
